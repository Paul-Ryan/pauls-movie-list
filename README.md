# Paul's Movie List

A curated list of Paul's favorite movies and TV shows, hosted on GitHub Pages.

## Pages

- **index.html** — The main movie and TV show list
- **logo.html** — Site logo/banner
- **ads.html** — Branding concepts and tagline variations

## Development

This is a static site with no build step. Edit the HTML files directly and push to `main` to deploy via GitHub Pages.
